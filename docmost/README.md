## Getting started
To get started with Pyjri, please refer to our [documentation](https://Pyjri.com/docs).

## Features
- Real-time collaboration
- Diagrams (Draw.io, Excalidraw and Mermaid)
- Spaces
- Permissions management
- Groups
- Comments
- Page history
- Search
- File attachment
